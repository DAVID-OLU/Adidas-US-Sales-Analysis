# Adidas-US-Sales-Analysis
Adidas US Sales Dashboard: Revenue, Retailer Performance, Product Insights &amp; Regional Trends

## 🎯 Objective
To analyze Adidas US sales data and uncover key business insights related to revenue performance, product category demand, regional sales distribution, and profit trends using an interactive dashboard.

## ⚙️ Methodology

Data Preparation

Cleaned and standardized raw sales dataset

Handled missing values and inconsistent product naming

Converted date fields into Year, Quarter, and Month format

Created calculated fields such as Total Revenue, Profit, and Margin

Data Modeling

Structured the dataset for analysis by:


Time (Year / Quarter / Month)
Product Category (Shoes, Apparel, Accessories, etc.)

Retailer / Region

Sales Metrics (Units Sold, Revenue, Profit, Margin)
